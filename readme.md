## Installation:
1. Download `installer.bat` and launch

## Usage
    cattools [OPTIONS]
## General Options:
    help, h                     Prints commands and options
    version, v                  Prints program version
    sinfo                       Prints system info
    whereis, where FILENAME     Finds path to file searched for

## Functions/Uses cases:
- System Info
- Find files
